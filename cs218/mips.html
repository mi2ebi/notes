RISC architecture

Qtspim - emulator thingy

x86 is written in CISC (complete instruction set computation)

MIPS is in RISC (r stands for reduced)

RISC has ~100 instructions compared to CISC's >1000
eg no inc, just add 1 urself smh
RISC is actually faster though

avg CISC instruction takes 4 lock cycles; 1 cycle/instr for RISC

data sizes

mips is 32 bits

byte - 8 bits. @s are still byte addressable but only 32 bits long

word - 32 bits! what the fuck

halfword - 16 bits

float - 32 bits
double - 64 bits

registers

name # purpose
$zero 0 hardware zero
$at 1 assembler temp
$v0 $v1 2-3 return values
$a0-$a3 4-7 function args
$t0-$t9 8-15, 24-25 temporary (not preserved)
$s0-$s7 16-23 saved registers
$sp 29 stack ptr
$fp 30 frame ptr (rbp)
$ra 31 return addr of previous fn
$pc n/a program counter (rp)
$h $l n/a used for 64 bit stuff
